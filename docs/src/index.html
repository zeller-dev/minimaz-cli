Minimaz CLI

Minimal input, clean output ⚡

About

Minimaz is a minimal, low-dependency static site builder and project initializer focused on speed, simplicity, and clean output. It supports HTML, CSS, JS, and TypeScript — compile, bundle, and minify with minimal configuration. Ideal for small static sites, utility projects, and quick prototypes.

Key Features

  • Initialize projects from templates
  • Build and minify HTML, CSS, JS, and TypeScript
  • Optional bundling of CSS/JS files
  • Include external styles/scripts from any path
  • Path replacements for asset links
  • Interactive CLI with timeout prompts
  • Optional npm and Git initialization
  • Lightweight, fast, and low-dependency

Use Cases

  • Small static websites and landing pages
  • Utility projects and internal tools
  • Portfolios and documentation sites
  • Teams needing reproducible minimal build systems

Requirements

  • Node.js ≥ 22
  • npm (optional if using npx)

Installation

Run directly with npx — no global install required.

npx minimaz init my-site
cd my-site
npx minimaz build
npx minimaz version

Aliases:

npx mz i my-site
npx mz b
npx mz v

Basic Workflow

1. Initialize 

npx minimaz init my-site
# or
npx mz i my-site

Creates a new project from a template. Supports npm and Git initialization with optional providers.

2. Build 

npx minimaz build
# or
npx mz b

Generates the dist/ folder with minified and optionally bundled assets.

3. Clear 

npx minimaz clear
# or
npx mz c

Deletes the dist/ folder to start fresh.

Command Summary

Available Minimaz CLI commands and aliases
Command Alias Description
minimaz init <name> [--template <name>] [--npm] [--git] [--gitprovider <url>] mz i Create a new project with optional template, npm, git, and provider settings
minimaz build mz b Build and minify the project
minimaz clear mz c Clear the dist/ folder
minimaz template <path> mz t Save current folder as a template
minimaz template --list mz t -l List available templates
minimaz template --delete <name> mz t -d Delete a template
minimaz template --update [<name>] mz t -u Update one or all templates
minimaz version mz v Show Minimaz CLI version
minimaz help [<command>] mz h Show help for CLI or specific command
