Minimaz CLI

Minimal development, maximum result ⚡

About

Minimaz is an open-source command-line tool built on Node.js. It reduces front-end complexity and makes static site publishing faster and cleaner. Ideal for portfolios, business pages, landing pages, and technical documentation.
This very page was generated with Minimaz!

Use Cases

  • Developers who need a simple static site workflow.
  • Designers who want to create and deploy clean websites fast.
  • Teams aiming for reproducible and minimal build systems.
  • Anyone who prefers zero-dependency, fast CLI tools.

Requirements

  • Node.js ≥ 22
  • npm ≥ 11

Installation

npm install -g minimaz-cli
minimaz version
# or using alias
mz v

The installer creates a ~/.minimaz directory containing templates and settings used by the CLI.

Basic Workflow

Step 1 – Initialize

minimaz init my-project -template default
# or
mz i my-project -t default

Creates a new static site structure using the default or a custom global template.

Step 2 – Manage Templates

mz template               # Save current folder as a template
mz template -update       # Update built-in templates
mz template -delete name  # Remove a specific template

Step 3 – Build

minimaz build
# or
mz b

Generates the dist/ folder with minified HTML, CSS, and JS. Best used in CI/CD pipelines for automatic deploys.

Command Summary

Command Alias Description
minimaz init <project> mz i Initialize a new project from template
minimaz build mz b Build and optimize the project
minimaz template <path> mz t Save a folder as template
minimaz template -l mz t -l List templates
minimaz template -d <name> mz t -d Delete a template
minimaz help mz h Display help
minimaz version mz v Show version

Configuration Example

{
  "src": "src",
  "dist": "dist",
  "public": "public",
  "minify": { "html": true, "css": true, "js": true },
  "replace": { "../public/": "public/" },
  "styles": [ "style.css" ],
  "scripts": [ "script.js" ]
}

The configuration file minimaz.config.json controls the build behavior and file handling.
