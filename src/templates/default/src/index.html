🚀 Welcome to Minimaz

Minimaz is a minimal static site builder designed for speed and simplicity. It helps you:

  • 💡 Organize your files with a simple structure
  • ⚙️ Automatically build and minify HTML, CSS, and JS
  • 🔄 Use a flexible minimaz.config.json file for configuration

🔧 Getting Started

Install Minimaz and create your first project:

# create a new project
npx minimaz init my-site

# build the site
npx minimaz build

📁 Project Structure

my-site/
├── src/               # HTML, CSS, JS files
├── public/            # Static assets (images, fonts, etc.)
├── dist/              # Output folder (generated)
├── minimaz.config.json
└── ...

⚙️ Configuration

The minimaz.config.json file lets you control the build process:

{
  "src": "src",
  "dist": "dist",
  "public": "public",
  "minify": {
    "html": true,
    "css": true,
    "js": true
  },
  "replace": {
    "../public/": "public/"
  }
}

❓ Help Command

Show the help menu with:

npx minimaz help

You're ready to build fast and clean static sites. Happy coding! 🛠️
