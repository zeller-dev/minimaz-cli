🚀 Welcome to Minimaz

Minimaz is a minimal static site builder designed for speed and simplicity. It helps you:

  • 💡 Organize your files with a simple structure
  • ⚙️ Automatically build and minify HTML, CSS, JS, and TS
  • 🔄 Customize file inclusion order using minimaz.config.json

🔧 Getting Started

Install and use Minimaz with the following commands:

# 🔨 Initialize a new project
npx minimaz init my-site

# 🏗️ Build the site
npx minimaz build

# 📦 Save template or list/delete existing ones
npx minimaz template <path> [--list|-l] [--delete|-d]

# 💬 Show help guide
npx minimaz help

📁 Project Structure

my-site/
├── src/                  # Source files (HTML, CSS, JS, TS)
├── public/               # Static assets (images, fonts, etc.)
├── dist/                 # Output directory (auto-generated)
└── minimaz.config.json   # Configuration file

⚙️ Configuration

{
  "src": "src",
  "dist": "dist",
  "public": "public",
  "minify": { "html": true, "css": true, "js": true, "ts": true },
  "replace": { "../public/": "public/" },
  "styles": [ "reset.css", "style.css", "theme.css" ],
  "scripts": [ "libs/jquery.js", "utils.js", "script.js" ]
}

📤 Production

The final output will be generated in the dist/ folder, ready for deployment.

❓ Available Commands

  • init / i – Initialize a new project
  • build / b – Build and minify the project
  • help / h – Show the help message
  • template / t – Manage templates (with --list and --delete options)

You're ready to build fast and clean static websites. Happy coding! 🛠️
