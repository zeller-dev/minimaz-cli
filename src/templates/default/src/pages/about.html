About Minimaz

Minimaz is a minimal build tool that helps you quickly scaffold and minify HTML, CSS, and JavaScript projects.

Features

  • Zero-config HTML/CSS/JS minification
  • Folder structure based on src, public, and dist
  • Simple CLI with init, build, and help commands

Getting Started

$ minimaz init my-project
$ cd my-project
$ minimaz build

Learn more or contribute on GitHub.
